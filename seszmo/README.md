# Sezmo Multipage Portfolio

Dark-only static portfolio for Sezmo with separate pages for each major section.

## Pages

- `index.html` - hero page
- `about.html` - bio and skill meters
- `showcase.html` - animated responsive showcase with 3D project movement
- `services.html` - service cards
- `featured.html` - featured cinematic project
- `testimonials.html` - client feedback
- `contact.html` - contact details and form

Open `index.html` in a browser to start the site.
